# 标准集成方案

本章节提供轻易云 iPaaS 官方标准集成方案包，涵盖主流业务场景，可直接导入使用。

## 方案列表

| 方案 | 适用场景 | 复杂度 |
|------|----------|--------|
| [跨境电商方案](../solutions/crossborder-ecommerce.md) | 跨境电商平台-ERP-海外仓集成 | 高 |
| [国内电商方案](../solutions/retail.md) | 淘宝/京东/拼多多-ERP-WMS 集成 | 中 |
| [OA 协同方案](../solutions/oa-finance.md) | 钉钉/飞书-ERP/HR 审批流集成 | 中 |
| [CRM 集成方案](../solutions/crm-integration.md) | CRM-ERP-财务系统客户主数据同步 | 中 |
| [ERP 对接方案](../solutions/mes-erp.md) | 多 ERP 间基础资料与单据同步 | 高 |
| [MES 集成方案](../solutions/mes-erp.md) | ERP-MES-设备数据采集集成 | 高 |
| [WMS 集成方案](../solutions/logistics.md) | ERP-WMS-物流平台集成 | 中 |
| [iPaaS Lite 方案](../quick-start/first-integration.md) | 轻量级数据同步场景 | 低 |

## 使用方式

1. 登录轻易云 iPaaS 控制台
2. 进入「方案市场」-「标准方案」
3. 选择对应方案点击「导入」
4. 根据向导配置连接信息
5. 启用并测试方案

> [!TIP]
> 标准方案提供 30 天免费试用，满意后再决定是否购买正式授权。
